Сохранение в текстовый файл и файл Excel таблиц адресов 101 и 104 пртокола из резервного файла МТ-500.
(package.tar.gz/package.tar/etc/KC/iec104_serv.xml и iec101_req.xml)

Так же возможно создание таблицы адресов из файла XML.

Python 2.7.14
Используемые библиотеки:
lxml 3.4.4
xlwt 1.1.2

Сборка в EXE файл:
PyInstaller 3.3
